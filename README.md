# University-Management-System-Java

A comprehensive console-based University Management System developed using Java, Object-Oriented Programming (OOP), JDBC, and MySQL. This project is designed to simulate a real-world academic management environment where administrators can efficiently manage students, authentication systems, and database operations through a modular and scalable architecture.

The system implements role-based authentication and follows a structured layered design using packages such as model, dao, util, and main to maintain clean code organization and separation of concerns. JDBC connectivity is used to establish communication between the Java application and MySQL database, enabling dynamic CRUD operations and secure data handling.
